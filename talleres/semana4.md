# Taller: Hoja de Vida en React

### Objetivo
Reconstruir tu hoja de vida (de la Semana 1) como una **aplicación React** con componentes.
Transformar la hoja de vida en una aplicación interactiva se pueda gestionar sus propias secciones de contenido (ej. experiencia, estudios, habilidades) mediante operaciones CRUD, simulando un panel de administración personal.

### Requisitos
1. **Al menos 4 componentes**:
   - `Header` → nombre, foto
   - `DatosPersonales` → contacto, hobbies
   - `Estudios`
   - `Experiencia`
2. **Uso de props** para personalizar cada sección
3. **Un estado interactivo** (ej. mostrar/ocultar, cambio de tema)
4. **Estilos CSS básicos** (archivo separado o en línea)
5. **Agregar efecto parallax en el background**

Usar como referencia
https://docs.rxresu.me/
https://rxresu.me/

**Puntos extra:** Aplicar persistencia con localstorage.

### Entregable

- Aplicación React funcional (npm run dev) en repositiorio de github
- Al menos una sección con CRUD completo (ej. experiencia laboral)
- Persistencia con localStorage (para que los datos no se pierdan al recargar)
- Interfaz limpia (puede usar Tailwind o CSS puro)
- Código modular (componentes separados: FormularioExperiencia, ListaExperiencias, etc.)
  
2. Reflexión escrita (200 palabras):  
   > “¿Qué ventajas y desafíos encuentras al usar React frente al desarrollo con HTML/CSS/JS puro?”
