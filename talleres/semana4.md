# Taller: Hoja de Vida en React

### Objetivo
Reconstruir tu hoja de vida (de la Semana 1) como una **aplicación React** con componentes.

### Requisitos
1. **Al menos 4 componentes**:
   - `Header` → nombre, foto
   - `DatosPersonales` → contacto, hobbies
   - `Estudios`
   - `Experiencia`
2. **Uso de props** para personalizar cada sección
3. **Un estado interactivo** (ej. mostrar/ocultar, cambio de tema)
4. **Estilos CSS básicos** (archivo separado o en línea)

### Entregable
- Funcional con `npm run dev`
- Código limpio y comentado
  
2. Reflexión escrita (200 palabras):  
   > “¿Qué ventajas y desafíos encuentras al usar React frente al desarrollo con HTML/CSS/JS puro?”
